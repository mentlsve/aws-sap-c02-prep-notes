# S3

* Data sits in a single region, but namespace is global
* Max file size 5TB
* Standard, Standard IA, One Zone IA, Intelligent Tiering
* Transitioning waterfall
    * Standard
    * Standard IA
    * Intelligent-Tiering
    * One-Zone IA
    * Glacier Instant
    * Glacier Flexible
    * Glacier Deep Archive
* Versioning needs to be enabled for replication
